     git push -u origin main

2. CONFIGURE SECRETS
   - Follow GITHUB_SECRETS.md
   - Add: HETZNER_TOKEN, SSH_PUBLIC_KEY

3. DEPLOY INFRASTRUCTURE
   - Run "Provision K3s Cluster" workflow (action: apply)
   - Wait 5-10 minutes

   - Run "Bootstrap Flux" workflow
   - Wait 3-5 minutes

TROUBLESHOOTING QUICK REFERENCE
================================================================================

Check cluster health:
  kubectl get nodes
  kubectl get pods -A

Check Flux status:
  flux get all -A
  flux logs

Check certificates:
  kubectl get certificate -A
  kubectl describe certificate -n <namespace> <cert-name>

Check application logs:
  kubectl logs -n wallabag -l app=wallabag
  kubectl logs -n linkding -l app=linkding

Destroy infrastructure:
  cd terraform/environments/demo
  terraform destroy
